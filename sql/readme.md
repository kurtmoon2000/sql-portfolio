# SQL Scripts

This folder contains SQL scripts used to build, populate, and query the database for my healthcare analytics projects.  
All scripts were written and executed in Azure SQL Database and organized to reflect a real analyst workflow.

## Included Scripts
- **create_database.sql** – Creates the database environment for public health datasets.
- **create_tables.sql** – Defines table structures for national vital statistics and related datasets.
- **insert_national_data.sql** – Loads cleaned public health data into the database.
- **queries.sql** – Analytical queries used for exploration, validation, and Power BI data modeling.

## Purpose
These scripts demonstrate:
- Database design and schema creation  
- Data loading and transformation  
- Writing analytical SQL queries  
- Supporting Power BI dashboards with optimized SQL views
