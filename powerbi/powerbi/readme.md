
# Power BI Dashboards

This folder contains Power BI reports developed for my healthcare analytics portfolio.  
Each dashboard is built using SQL-based datasets and focuses on public health, clinical metrics, and data storytelling.

## Included Reports
- **Vital Statistics Dashboard** – Analysis of national birth and mortality trends.
- **Fetal Mortality Dashboard** – Visual exploration of fetal mortality rates and contributing factors.

## Purpose
These dashboards demonstrate:
- Data modeling and transformation
- DAX calculations
- Visual design and layout
- Healthcare-focused analytics and insights
